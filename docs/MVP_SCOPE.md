# MVP Scope

The MVP scope is locked to the following feature areas:

1. Authentication and role-based access control
2. Patient registration
3. Patient search
4. Patient history
5. Doctor management
6. Appointment scheduling with slot conflict prevention
7. Basic EMR and consultation notes
8. AI chatbot for symptom pre-consultation
9. Voice-to-text consultation note support
10. E-prescription generation
11. Billing and invoice generation
12. Admin dashboard
13. Audit logs
14. Dockerized local development
15. API documentation
16. Basic tests

## Explicitly Out Of Scope For Phase 0

- Patient workflows
- Authentication flows
- Appointment logic
- Billing logic
- Production deployment automation
- Advanced AI clinical decision support

## Foundation Stack Note

The current frontend and backend foundations use JavaScript rather than TypeScript:

- Frontend: React + Vite + JavaScript
- Backend: Node.js + Express.js + JavaScript
